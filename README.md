# 🎓 AceSched - 鸡娃卷

> 家长课程管理助手，高效管理孩子的课外课程、课时、费用和日程

[![Android](https://img.shields.io/badge/Android-1.78.0-green.svg)](https://github.com/ouosavey/acesched-releases/releases)
[![HarmonyOS](https://img.shields.io/badge/HarmonyOS-2.00.11-blue.svg)](https://github.com/ouosavey/acesched-releases/releases)

## ✨ 功能特性

- 📚 **课程管理** — 添加/编辑/归档课程，跟踪课时进度和费用
- 👨‍👩‍👧 **多成员支持** — 一个账号管理多个成员，切换查看各自课程
- 👨‍👩‍👧‍👦 **家庭模式** — 家庭成员共享数据，权限分级管理
- 📅 **课程日历** — 月/周视图查看上课安排，支持请假和调课
- 📎 **课程资料** — 上传课程资料到云端，应用内预览图片/PDF
- 📥 **课程导入** — 扫描二维码或选择家庭成员，一键导入课程
- 🔄 **日历同步** — 一键同步到系统日历，自动创建专用日历
- 📊 **统计报表** — 年度花费、课时进度、费用明细等可视化统计
- 🌙 **深色模式** — 支持白天/夜晚/跟随系统三种主题
- 🔔 **消息提醒** — 上课前推送通知提醒

## 📱 下载安装

前往 [Releases](https://github.com/ouosavey/acesched-releases/releases) 页面下载最新版本。

### Android

| 架构 | 适用设备 | 文件 |
|------|----------|------|
| arm64-v8a | 主流手机（推荐） | `app-arm64-v8a-release.apk` |
| armeabi-v7a | 老旧32位手机 | `app-armeabi-v7a-release.apk` |
| x86_64 | 模拟器/x86设备 | `app-x86_64-release.apk` |

> 大多数用户请下载 `arm64-v8a` 版本。覆盖安装即可，无需卸载旧版（除非 Release 说明中特别标注）。

### HarmonyOS

下载 `.hap` 文件，通过 DevEco Studio 或 hdc 命令安装到鸿蒙设备。
